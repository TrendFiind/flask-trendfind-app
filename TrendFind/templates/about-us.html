Our Story

A Bedroom, A Laptop, A Hunch

It began with a restless experiment: "What if GPT could rank eBay listings by resale odds?" One messy Python file later, I’d flipped a phone case for a 42 % margin. Validation in 7 days.

The code ballooned; APIs throttled; caffeine vanished. Friends joined private betas. Profits climbed. The script needed a name—TrendFind stuck.

# first_prototype.py  –  May 2023\...
import requests, pandas as pd
from openai import ChatCompletion

feed = requests.get('https://fakestoreapi.com/products').json()
df   = pd.DataFrame(feed)[['title','price','rating']]

prompt = f"Rank these {len(df)} products by resale potential." 
resp   = ChatCompletion.create(model='gpt-4o', messages=[{'role':'user','content':prompt}])
print(resp.choices[0].message.content)
