How TrendFind Was Born

A Spark in a Student Bedroom

I was 19, juggling uni projects and late-night rabbit holes in AI. One weekend, I bolted a simple Python scraper onto an open-source GPT model. The script spat out three product ideas—one of which flipped for a 42 % margin on eBay the next week. TrendFind’s seed was planted.

That hacky script matured line by line: learning REST calls, wrangling JSON, throttling APIs. The more code I wrote, the clearer the vision became: **turn data chaos into seller clarity**.

import requests, pandas as pd
from openai import ChatCompletion

feed = requests.get('https://fakestoreapi.com/products').json()
df = pd.DataFrame(feed)[['title','price','rating']]

prompt = f'Rank these {len(df)} products by resale potential.'
resp = ChatCompletion.create(model='gpt-4o', messages=[{'role':'user','content':prompt}])
print(resp.choices[0].message.content)
